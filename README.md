# NABP (NamuICT Baremetal Platform)

**NABP** is a bare-metal provisioning and lifecycle management platform developed by **NamuICT**.  
It is designed to automate OS provisioning, proxy deployment, and infrastructure integration
around Foreman-based environments.

This repository serves as the **official entry point** for NABP source distribution.
